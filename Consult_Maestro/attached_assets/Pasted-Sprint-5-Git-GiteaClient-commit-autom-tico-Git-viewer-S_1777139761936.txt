ENTREGAR:

1. server/infra/giteaClient.ts (arquivo novo)
   Classe GiteaClient com métodos:
   
   createRepo(name, description)
     → POST /api/v1/user/repos
     → private: true, auto_init: true, default_branch: 'main'
   
   commitFile(owner, repo, filePath, content, message, branch?)
     → Verificar SHA existente antes (GET contents)
     → PUT se arquivo existe (com sha), POST se novo
     → content em base64
   
   listCommits(owner, repo, branch?)
     → GET /api/v1/repos/{owner}/{repo}/commits
   
   getCommitDiff(owner, repo, sha)
     → GET /api/v1/repos/{owner}/{repo}/git/commits/{sha}
     → Retornar array de arquivos com patch
   
   createBranch(owner, repo, branchName, fromBranch?)
     → POST /api/v1/repos/{owner}/{repo}/branches
   
   getFileContent(owner, repo, path, branch?)
     → GET /api/v1/repos/{owner}/{repo}/contents/{path}
     → Retornar conteúdo decodificado de base64
   
   Factory getGiteaClient(tenantId):
     → Ler URL e token do Gitea em infra_servers
       onde serviceType = 'gitea'
     → Mesmo padrão de criptografia do CoolifyClient

3. INTEGRAÇÃO no orchestrator — server/ide/orchestrator.ts
   Adicionar em executeDeploy() APÓS deploy bem-sucedido:
   
   a) Criar ou reutilizar repositório do projeto
      repoName = 'project-' + projectId
      try { await gitea.createRepo(repoName, designDoc.summary) }
      catch {} // idempotente — não falha se já existe
   
   b) Commitar cada artefato gerado:
      Para cada doctype:
        gitea.commitFile('arcadia', repoName,
          'doctypes/{name}.json', doctype.json,
          'feat(doctype): {name} — Agente Developer')
      
      Para cada server_script:
        gitea.commitFile('arcadia', repoName,
          'scripts/{name}.py', script.code,
          'feat(script): {name} — QA PASS')
      
      Design Doc:
        gitea.commitFile('arcadia', repoName,
          'docs/design-doc.json',
          JSON.stringify(designDoc, null, 2),
          'docs: Design Document — ' + designDoc.summary)
   
   c) Salvar gitRepoUrl no projeto:
      db.update(devProjects)
        .set({ gitRepoUrl: GITEA_URL + '/arcadia/' + repoName })
   
   d) Se Gitea não configurado: silencioso — não quebrar o deploy
