## Como aplicar no Replit

Copie a pasta `patches/` para a raiz do projeto e execute **na ordem abaixo**:

```bash
# Na raiz do projeto Arcádia Consult:
cd /path/to/arcadia-consult

# 1. MT-1 estendida + Grupo C (routes.ts)
python3 patches/01_mt1_extended_grupo_c.py

# 2. MT-2 — UI Superadmin (novos arquivos frontend)
# Copiar os arquivos gerados manualmente (ver instruções no patch)
node patches/02_mt2_superadmin_ui.js

# 3. MT-3 — Multi-empresa (schema + storage + routes + UI)
python3 patches/03_mt3_multi_empresa.py

# 4. MT-4 — LLM badge + uso por empresa
node patches/04_mt4_llm_ux.js

# Após todos:
npm run build  # ou reiniciar no Replit
```
